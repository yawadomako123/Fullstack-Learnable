## Anweisungen

Jetzt, wo Sie ein ARIMA-Modell erstellt haben, bauen Sie ein neues mit frischen Daten (versuchen Sie eines von [diesen Datensätzen von Duke](http://www2.stat.duke.edu/~mw/ts_data_sets.html)).
