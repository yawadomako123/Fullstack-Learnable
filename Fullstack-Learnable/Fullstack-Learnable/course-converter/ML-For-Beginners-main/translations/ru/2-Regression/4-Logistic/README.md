1. –ü–æ—Ä—è–¥–∫–æ–≤—ã–π –∫–æ–¥–∏—Ä–æ–≤—â–∏–∫: –æ–Ω —Ö–æ—Ä–æ—à–æ –ø–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è –ø–æ—Ä—è–¥–∫–æ–≤—ã—Ö –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö, –∫–æ—Ç–æ—Ä—ã–µ —è–≤–ª—è—é—Ç—Å—è –∫–∞—Ç–µ–≥–æ—Ä–∏–∞–ª—å–Ω—ã–º–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–º–∏, –≥–¥–µ –∏—Ö –¥–∞–Ω–Ω—ã–µ —Å–ª–µ–¥—É—é—Ç –ª–æ–≥–∏—á–µ—Å–∫–æ–º—É –ø–æ—Ä—è–¥–∫—É, –∫–∞–∫ —Å—Ç–æ–ª–±–µ—Ü `Item Size` –≤ –Ω–∞—à–µ–º –Ω–∞–±–æ—Ä–µ –¥–∞–Ω–Ω—ã—Ö. –û–Ω —Å–æ–∑–¥–∞–µ—Ç –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏–µ, —Ç–∞–∫ —á—Ç–æ –∫–∞–∂–¥–∞—è –∫–∞—Ç–µ–≥–æ—Ä–∏—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∞ —á–∏—Å–ª–æ–º, –∫–æ—Ç–æ—Ä–æ–µ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç –ø–æ—Ä—è–¥–∫—É –∫–∞—Ç–µ–≥–æ—Ä–∏–∏ –≤ —Å—Ç–æ–ª–±—Ü–µ.

    ```python
    ordinal_encoder = OrdinalEncoder(categories=item_size_categories)
    ```

2. –ö–∞—Ç–µ–≥–æ—Ä–∏–∞–ª—å–Ω—ã–π –∫–æ–¥–∏—Ä–æ–≤—â–∏–∫: –æ–Ω —Ö–æ—Ä–æ—à–æ –ø–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è –Ω–æ–º–∏–Ω–∞–ª—å–Ω—ã—Ö –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö, –∫–æ—Ç–æ—Ä—ã–µ —è–≤–ª—è—é—Ç—Å—è –∫–∞—Ç–µ–≥–æ—Ä–∏–∞–ª—å–Ω—ã–º–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–º–∏, –≥–¥–µ –∏—Ö –¥–∞–Ω–Ω—ã–µ –Ω–µ —Å–ª–µ–¥—É—é—Ç –ª–æ–≥–∏—á–µ—Å–∫–æ–º—É –ø–æ—Ä—è–¥–∫—É, –∫–∞–∫ –≤—Å–µ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∏, –æ—Ç–ª–∏—á–Ω—ã–µ –æ—Ç `Item Size` –≤ –Ω–∞—à–µ–º –Ω–∞–±–æ—Ä–µ –¥–∞–Ω–Ω—ã—Ö. –≠—Ç–æ –æ–¥–Ω–æ—Ä–∞–∑–æ–≤–æ–µ –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ, —á—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –∫–∞–∂–¥–∞—è –∫–∞—Ç–µ–≥–æ—Ä–∏—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∞ –±–∏–Ω–∞—Ä–Ω—ã–º —Å—Ç–æ–ª–±—Ü–æ–º: –∑–∞–∫–æ–¥–∏—Ä–æ–≤–∞–Ω–Ω–∞—è –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è —Ä–∞–≤–Ω–∞ 1, –µ—Å–ª–∏ —Ç—ã–∫–≤–∞ –ø—Ä–∏–Ω–∞–¥–ª–µ–∂–∏—Ç —ç—Ç–æ–π —Ä–∞–∑–Ω–æ–≤–∏–¥–Ω–æ—Å—Ç–∏, –∏ 0 –≤ –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ.
