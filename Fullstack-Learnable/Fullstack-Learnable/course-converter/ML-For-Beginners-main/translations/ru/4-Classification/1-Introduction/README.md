1. –°–æ–∑–¥–∞–π—Ç–µ —Ñ—É–Ω–∫—Ü–∏—é `create_ingredient()` –Ω–∞ Python –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –¥–∞—Ç–∞—Ñ—Ä–µ–π–º–∞ –∏–Ω–≥—Ä–µ–¥–∏–µ–Ω—Ç–æ–≤. –≠—Ç–∞ —Ñ—É–Ω–∫—Ü–∏—è –Ω–∞—á–Ω–µ—Ç —Å —É–¥–∞–ª–µ–Ω–∏—è –Ω–µ–Ω—É–∂–Ω–æ–≥–æ —Å—Ç–æ–ª–±—Ü–∞ –∏ –æ—Ç—Å–æ—Ä—Ç–∏—Ä—É–µ—Ç –∏–Ω–≥—Ä–µ–¥–∏–µ–Ω—Ç—ã –ø–æ –∏—Ö –∫–æ–ª–∏—á–µ—Å—Ç–≤—É:

    ```python
    def create_ingredient_df(df):
        ingredient_df = df.T.drop(['cuisine','Unnamed: 0']).sum(axis=1).to_frame('value')

        return ingredient_df
    ```
