## Öğrenecekleriniz

Bu bölümde, önceki Regresyon çalışmanız üzerine inşa edecek ve verileri daha iyi anlamak için kullanabileceğiniz diğer sınıflandırıcılar hakkında bilgi edineceksiniz.
