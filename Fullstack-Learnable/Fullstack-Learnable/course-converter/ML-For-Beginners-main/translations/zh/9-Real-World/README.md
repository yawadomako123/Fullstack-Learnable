# 后记：经典机器学习的实际应用

在本课程的这一部分中，你将了解经典机器学习的一些实际应用。我们在互联网上搜寻了白皮书和文章，介绍了使用这些策略的应用，尽量避免涉及神经网络、深度学习和人工智能。了解机器学习在商业系统、生态应用、金融、艺术和文化等方面的应用。
